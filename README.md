# Plateforme-numerique-ecole

Cette plateforme a vue le jour lors du premier confinement en mars 2020. 
Pour faire face aux difficultés rencontrées par l'école pour "l'école à la maison", une solution web était la meilleure option.
Ainsi, tous les échanges sont centralisés à un seul et même endroit.


Plateforme réalisée sous Symfony 5, en autonomie (avec échanges réguliers avec l'équipe enseignante).
Cette réalisation a été faite 100% sur mesure, elle ne pourrait donc pas convenir à tous les établissements scolaires.
